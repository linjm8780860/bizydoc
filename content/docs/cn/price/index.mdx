# BizyAir 计费简介

## 一、计费方式

根据 **服务器上真实推理时长（不包括排队、网络传输时间）** 扣除 BZ 币。不做特殊说明的情况下，**每秒扣除5个BZ币**。个别任务以次数计费（如 LoRA 训练）扣除对应的 BZ 币。具体可见 [节点价格参考](./price)。

## 二、账单查询

- 可以在个人资料（Profile）中点击充值记录（Payment）查看充值记录。
- 可以在个人资料（Profile）中点击个人账单（Bill）查看消费记录。

## 三、BZ 币

BizyAir 向用户提供 “BZ币” 虚拟产品，授权用户使用包括但不限于 BizyAir 节点服务、任务执行等平台功能。

BZ 币分为两类：

   - **金币**：用户通过充值直接购买的BZ币；
   - **银币**：平台通过活动、会员权益等途径赠予的BZ币。

BizyAir 部分云节点和任务，只能使用“金币”支付，不能使用“银币”支付。

其他更详细的说明，请参考 [充值服务协议](../terms/terms)。
