# 分享直达码

模型广场和工作流广场的模型、工作流，都有对应的直达码，用户通过直达码，可以直接跳转到对应模型、工作流的详情页。

## 获取模型的直达码

在模型广场，点击模型，可以弹出模型的详情页。在详情页，点击分享按钮 :分享: 可以得到直达码（系统会提示 “The share code has been copied!”）。
直达码会直接复制在你的剪切板里，你可以粘贴发送给你的朋友。

## 通过直达码找到模型详情页

用户得到直达码后，可以在 BizyAir 工具栏中通过直达码直接跳转到模型的详情页。具体方法如下：

鼠标悬浮在 BizyAir 工具栏的 :礼包: 时，会显示输入框，将直达码填入输入框后，点击 :兑换: 按钮，即可弹出详情页。
