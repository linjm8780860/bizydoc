# 模型广场

模型广场用于交流和使用模型。

## 上传模型

在 BizyAir 的工具栏中点击 :发布: Publish 菜单后，选择 Model 可以发布模型。

在接下来的对话框中，在 Model Name 输入栏中填写你要上传模型的名字，在 Model Type 输入栏中，选择你上次模型的类型（比如是 LoRA 还是 ControlNet）。
然后点击 “Next Step” 下一步。

在接下来的对话框中，依次填写信息：

- Version Name：模型版本名，主要用于区分同一个模型下的不同模型版本，用户按照自己的喜好填写即可，如 "v1"。
- Base Model：模型生态所属的基础模型，如果有对应的选项，请选择之，否则，请选择 Other。
- Upload Image：可以上传模型有关的图片。
- Introduction：模型的介绍。
- Publicly Visible：是否公开，如果打开这个选项，其他用户可以在模型广场查看你上传的模型。

然后，点击或者拖拽文件至上传区域 :文件上传: ，即可上传模型。
上传完毕后，点击 "Publish" 可以发布。

## 使用模型

在 BizyAir 工具栏，点击 Community，在弹出的对话框中，选择 :我的模型: Contents 中，可以查看和使用自己上传的模型，以及 fork 自模型广场的其他模型。

点击 :加载模型:，可以得到一个该模型的节点。

## 复制模型

在模型广场中，点击模型的封面，可以弹出模型的详情页。在详情页中，点击 “Fork” 按钮，可以复制一份模型到自己的空间里。

## 编辑模型

在模型的详情页中，点击 :more: more 菜单后，选择 "Edit"，可以对模型重新进行编辑，保存后即可更新模型信息。
