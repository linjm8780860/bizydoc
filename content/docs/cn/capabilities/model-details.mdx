# 模型和工作流的详情页

在 [模型广场](model-plaza.md) 或者工作流广场点击模型或者工作流的封面，可以进入到模型或者工作流的详情页。

在详情页，以下几个功能值得关注：

- Fork：点击 Fork 可以复制公开的模型到自己的空间，Fork 后也可以 Unfork 还原。
- Add Node：在 BizyAir 中点击 Add Node，会生成一个加载该模型的 ComfyUI 节点，方便直接使用。
- :点赞: 点赞：如果喜欢该模型，可以给这个模型点赞。
- :分享: 分享：点击分享按钮，可以得到分享直达码，方便分享模型给其他人，具体可以参考 [分享直达码](sharecode.md)
