# 账号注册

## 如何注册

BizyAir账号分为两套不同的登陆体系

注册BizyAir账号：点击右上角 “登陆” - “注册”，按照提示即可完成注册。

注册SiliconFLow账号：右上角“登陆” - “SiliconFLow”，按照提示即可完成注册。
