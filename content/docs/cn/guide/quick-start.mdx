# 在线使用

在 [bizyair.cn](https://www.bizyair.cn) 网站在线使用，是目前最推荐的方式。无需本地配置，打开浏览器即可体验完整工作流，支持分享工作流、作品、发布 AI 应用。
你也可以通过安装 BizyAir 插件，使用 BizyAir 云端资源。

# 本地插件

以下几种方法均可安装 BizyAir，可以根据自己的实际情况选择 **其中一种** 最适合自己的方式。

## 使用 ComfyUI Manager

如果你已经有 ComfyUI，并且安装了 ComfyUI Manager，在 ComfyUI Manager 的搜索框中搜索 “BizyAir”，再点击 “Install” 即可完成安装。

## 通过 git 安装

如果你熟悉命令行操作，可以通过 git clone 的方式安装 BizyAir。

```bash
git clone https://gitcode.com/BizyAir/BizyAir.git
```

## 通过 Comfy CLI 安装

先安装 comfy-cli：

```bash
pip install comfy-cli
```

然后通过 comfy-cli 安装 BizyAir：

```bash
comfy node install bizyair
```
