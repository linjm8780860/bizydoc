# BizyAir 错误码说明

## 使用说明

- 错误由三部分组成：错误码（ErrCode）、HTTP 状态（HttpStatus）、错误消息（Message）。
- 一些错误码可能在不同业务场景复用，请结合 HTTP 状态与接口上下文理解。
---
